Lexa

Lexa is a browser extension that analyzes and highlights text directly in the user's browser. It processes content locally and provides real-time feedback without sending data to external servers.

Purpose

Lexa was created as a personal project to explore browser extension architecture, DOM manipulation, text analysis, and clean frontend engineering practices.

Features

* Real-time text analysis
* Highlighting by category
* Local processing (no external APIs required)
* Lightweight and fast

Tech Stack

* TypeScript
* Chrome Extensions (Manifest V3)
* DOM APIs
* Modular architecture (content / background / popup separation)

Installation (Development)

1. Clone the repository.
2. Run the build process.
3. Open Chrome and navigate to chrome://extensions.
4. Enable Developer Mode.
5. Click “Load unpacked” and select the build folder.

Privacy

Lexa does not collect, store, or transmit personal data. All processing happens locally within the user's browser.
